Nesta atividade iremos desenvolver um programa, utilizando as linguagens de programação Java ou Python, que irá permitir treinar os 
conceitos vistos no material teórico e nos encontros WEB.



Vamos desenvolver um sistema de cadastro de alunos e disciplinas utilizando o paradigma orientado a objetos. O programa deverá conter 
as seguintes classes:



Uma classe aluno, onde será criada contendo o nome do aluno, a matricula, idade e disciplinas associadas;


Uma classe chamada disciplinas, onde receberá os dados de nome da disciplina, alunos matriculados e o código;


Outras classes que o aluno julgar necessário para realizar gravação / manipulação dos dados e processamento;


Os alunos poderão estar associados a uma ou mais disciplinas. A classe disciplina será responsável por controlar o armazenamento das 
notas de todos os alunos;


O programa deverá permitir o cadastro de disciplinas e alunos. Após o cadastro o programa deverá permitir a atribuição de alunos as 
disciplinas solicitadas, alem das notas obtidas nas provas.


Cada aluno deverá receber duas notas para cada disciplina, das provas 1 e 2. Alunos com média inferior a 5 serão reprovados.Ao final 
do lançamento das notas, o programa deverá permitir a apresentação de um relatório, mostrando as notas das provas e a situação de 
cada aluno, por disciplina, no seguinte formato:



Disciplina 1 - Estatística - Código 2222

Mat Aluno N1 N2 Média Resultado

0101 Francisco 10 10 10 Aprovado



Disciplina 2 - Calculo 1 - Código 3333

Mat Aluno N1 N2 Média Resultado

0101 Francisco 10 10 10 Aprovado



O programa deverá fazer validações de entrada, não permitindo por exemplo a inclusão de notas negativas e campos vazios;

O código deverá utilizar Herança entre as classes.
